This project is an interactive Power BI dashboard built to analyze and visualize Kotak Mahindra Bank transaction data. The dashboard provides key financial insights through dynamic visualizations, helping users understand spending patterns, transaction categories, payment recipients, and yearly trends.

The dashboard summarizes transaction activity using KPIs, charts, and filters, enabling quick exploration of financial data and informed decision-making.

Key Features
Transaction Summary KPIs
Total Transaction Amount
Total Number of Transactions
Maximum Transaction Amount
Minimum Transaction Amount
Average Transaction Amount
Transaction Type Analysis
Breakdown of amounts by transaction types such as:
Paid
Top-up
Received
Time-Series Analysis
Track transaction amounts across Year, Quarter, Month, and Day.
Identify spending trends and transaction spikes over time.
Year-wise Distribution
Compare transaction amounts across different years.
Visual representation using a pie chart.
Customer Insights
Analyze transaction amounts by customer category.
Receiver Analysis
Treemap visualization showing the top receivers/payees.
Helps identify major spending destinations.
Interactive Filtering
Filter data by:
Transaction Type
Year
Enables focused and customized analysis.
Dashboard Benefits
Monitor financial transaction patterns.
Identify major payment categories and recipients.
Compare yearly transaction performance.
Gain insights into spending behavior through interactive visualizations.
Support data-driven financial analysis and reporting.
Tools Used
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Dashboard Preview

The dashboard provides a comprehensive overview of transaction activities through KPIs, bar charts, line charts, pie charts, treemaps, and slicers for interactive exploration.
